# cs293securestaticwebsite

Static webpage use for AWS S3 and CI/CD pipeline project.

## Technologies Used:
- AWS S3
- AWS IAM
- AWS EC2 (For Installing Jenkins)
- AWS CloudFront (For Secure Hosting and CDN)
- Jenkins or GitHub Actions
- GitHub
